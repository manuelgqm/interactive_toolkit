Set substitution criteria

Once intrinsic dangers of substances have been identified, the next step is to select priority substances for elimination which calls for alternative chemicals. Each organization sets its own criteria for substitution. Priority toxic properties to be avoided include:

  • Carcinogenicity
  • Reproductive toxicity
  • Long-term environmental effects
  • Neurotoxicity
